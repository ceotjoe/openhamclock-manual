# OpenHamClock Manual

The complete user guide for [OpenHamClock](https://openhamclock.com) — a modern, web-based amateur radio dashboard.

## Live Manual

👉 **[Read the manual](https://k0cjh.github.io/openhamclock-manual/)**

## Structure

Each section of the manual is a self-contained HTML page:

| File | Section |
|------|---------|
| `index.html` | Introduction |
| `quickstart.html` | Quick Start |
| `installation.html` | Installation |
| `configuration.html` | Configuration |
| `world-map.html` | World Map |
| `dx-cluster.html` | DX Cluster |
| `space-weather.html` | Space Weather & Band Conditions |
| `satellites.html` | Satellites |
| `pota-sota.html` | POTA / SOTA / WWFF / WWBOTA |
| `rig-bridge.html` | Rig Bridge |
| `wsjt-ft8.html` | WSJT-X / FT8 |
| `themes-profiles.html` | Themes & Profiles |
| `troubleshooting.html` | Troubleshooting |
| `reference.html` | Reference |
